# 🧠 Tasks & Notes Manager

A modern productivity app that combines intelligent task management and note organization into one seamless experience.  
Built with Flutter, Supabase, and FastAPI, featuring cloud sync, AI-powered reminders, and a clean customizable UI.

---

# ✨ Features

## 📝 Smart Notes Management

Organize your thoughts, ideas, and important information efficiently.

### Features:
- Create notes with:
  - Title
  - Content
- Pin important notes
- Notes are automatically sorted:
  - 📌 Pinned notes first
  - Regular notes afterward

Perfect for:
- Study notes
- Meeting points
- Daily journaling
- Quick reminders

---

## ✅ Intelligent Task Manager

Manage daily tasks with a simple yet powerful workflow.

### Task Features:
- Task title
- Completion status tracking
- Optional notifications
- Clean and distraction-free task interface

---

## 🤖 AI-Powered Personalized Notifications

A unique smart reminder system that generates notifications based on your preferred mood.

### Notification Moods:
- 😂 Funny
- 😏 Sarcastic
- 💪 Motivational
- 📌 Serious

Examples:
- *"Bro, your tasks aren't going to complete themselves."*
- *"Future you will either thank you or hate you."*
- *"One small task today = less stress tomorrow."*

---

## 🧍 Personalized Reminder Context

Users can optionally write about themselves to make AI reminders more personalized.

The system can adapt notifications based on:
- Goals
- Personality
- Habits
- Productivity style

Resulting in smarter and more engaging reminders.

---

## 🔔 Flexible Notification Scheduling

Supports multiple notification systems for better productivity planning.

### Notification Types:

#### 📅 Date-Based Notifications
Schedule reminders for specific dates and times.

#### 📆 Weekly Notifications
Choose repeating reminder days:
- Monday – Sunday
- Fully customizable weekly schedules
